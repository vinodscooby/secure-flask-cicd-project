# Secure Flask Web App with CI/CD & Security Scanning

**Project 1** - IBM Continuous Integration and Continuous Delivery (CI/CD)

---

## 🚀 Project Overview

A secure Python Flask REST API with fully automated **CI/CD pipeline** and multi-layered **security scanning** using GitHub Actions.

### Key Features
- Secure Flask Web Application
- End-to-End CI/CD Pipeline
- Automated Security Scanning (SAST, Dependencies, Secrets)
- Docker Containerization
- Unit Testing with pytest

## 🛠️ Technologies Used
- **Language**: Python 3
- **Framework**: Flask
- **CI/CD**: GitHub Actions
- **Security**: Bandit, pip-audit, Gitleaks
- **Container**: Docker
- **Testing**: pytest

## 📋 CI/CD Pipeline
- Code Checkout
- Dependency Installation
- Security Scans (Bandit + Dependency + Secret Scan)
- Unit Tests
- Docker Image Build
